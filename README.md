# MVVMTodo

## Overview, Purpose & Features
- it is a simple app which will show notes.
- user can create, edit & delete notes
- it has used all the best practices for development

## Concepts Used
- Navigation Component
- Recycler View
- Fragments
- Coroutines
- ViewModel && LiveData
- Room
- ViewBinding
- Dagger & Hilt
